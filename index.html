Spaceship Modular - Customizable Spaceship Generation Tool

Description: Spaceship Modular is a personal project developed by Bill The Dev, designed to help game developers easily create customized spaceship assets for their games.

This tool features 10 base spaceship models with 4 customizable parts: head, wings, engines, and weapons. These components allow for up to 10,000 unique spaceship variations. Additionally, users can modify colors, adjust metallic and surface properties, and craft polished, game-ready spaceships.

The project is a showcase of Bill The Dev’s 3D modeling skills and aims to contribute to the game development community by offering a powerful and user-friendly tool for creating custom spaceships.

Key Features:
- 10 base spaceship models.
- 4 customizable parts: head, wings, engines, and weapons.
- Over 10,000 possible spaceship variations.
- Customizable colors and material properties (metallics, gloss).
- Easy integration into game development projects.
